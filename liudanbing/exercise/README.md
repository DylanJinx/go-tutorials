1. 变量 
    - 变量的声明
    - 局部变量的声明
    - ⽅法⼀ //⽅法⼀：声明⼀个变量 默认的值是0
        var a int
    - ⽅法⼆ //⽅法⼆：声明⼀个变量，初始化⼀个值
        var b int = 100
    - ⽅法三 //⽅法三：在初始化的时候，可以省去数据类型，通过值⾃动匹配当前的变量的数据类型
        var c = 100
    - ⽅法四(常⽤) //⽅法四：(常⽤的⽅法) 省去var关键字，直接⾃动匹配
        e := 100
    - 全局变量的声明 ⽅法四不⽀持全局
    - 多变量的声明
        - 单⾏写法
            var xx, yy int = 100, 200
            var kk, ll = 100, "Aceld"
        - 多⾏写法
            var (
            vv int = 100
            jj bool = true
            )
